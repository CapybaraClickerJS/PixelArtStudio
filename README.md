# PixelArtStudio

Um editor de pixel art minimalista e offline-first para Android, desenvolvido completamente em uma aplicação web responsiva.

## ✨ Features

- 🎨 **Editor de Pixel Art** - Crie arte pixel diretamente no seu celular
- ✏️ **Ferramentas Completas** - Lápis, borracha, balde de tinta, seletor de cor
- 🔄 **Undo/Redo** - Desfaça e refaça ações ilimitadas
- 🔍 **Zoom** - Amplie e reduza conforme necessário
- 📥 **Importar PNG** - Carregue imagens PNG e edite
- ⬇️ **Exportar PNG** - Salve seus desenhos como PNG
- 💾 **Auto-save** - Seus desenhos são salvos automaticamente
- 🌙 **Dark Mode** - Interface moderna e escura
- ⚡ **Offline** - Funciona sem internet

## 📱 Compatibilidade

- Android 7.0+ (API 24+)
- Qualquer resolução de tela

## 🚀 Como Usar

1. Abra o app
2. Configure o tamanho da tela (32×32, 64×64, etc)
3. Escolha uma ferramenta (lápis, borracha, etc)
4. Desenhe!
5. Exporte como PNG
